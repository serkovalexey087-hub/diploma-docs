# 📚 Документация дипломного проекта
## Архитектура AI-агентов для обработки мультимедиа

### 🎯 Суть проекта
Создание методологии управления человеко-машинными системами через пайплайн независимых AI-агентов. Каждый агент принимает решения автономно и передаёт результат следующему этапу обработки.

### 📂 Содержимое репозитория

| Файл | Описание |
|------|----------|
| `process.bpmn` | BPMN-диаграмма полного цикла разработки и вывода продукта на рынок |
| `reglament.docx` | Детальный регламент выполнения процесса с ролями и ответственностью |

### 🔗 Интеграция с кодом
Полный код проекта находится в закрытом репозитории (в разработке до марта 2026). Документация открыта для демонстрации методологической экспертизы.
